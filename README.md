# Portfolio Website

A minimalist, easy-to-edit personal portfolio website.

## Edit content quickly

All personal details are stored in:

- `data/profile.js`

You can update your profile, work history, education, languages, links, and placeholder descriptions there without touching layout code.

## Project structure

- `index.html` - semantic page structure
- `styles.css` - minimalist visual design
- `script.js` - rendering logic from data
- `data/profile.js` - profile/content data source

## Run locally

Open `index.html` directly in your browser, or serve with a local server:

```bash
python3 -m http.server 8000
```

Then open <http://localhost:8000>.
